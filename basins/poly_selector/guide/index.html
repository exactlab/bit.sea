Poly selector User Guide

Overview

Poly selector user interface.

In the above figure you can see how the Poly selector tool presents itself when it is just loaded in the browser.

On the top left of the map there are the controls for zooming in and zooming out the view, start drawing a polygon, editing a drawn polygon (disabled because there are no polygons drawn) and deleting a polygon (also disabled).

Zoom, draw and edit controls.

On the top right of the map there is the layer control that allows to select the visible layers (see Map Layers below).

Layer control.

Exploring the map

The map can be dragged by clicking on the maps itself, holding the left mouse button and moving the cursor in the opposite direction (E.G. to explore the western portion of the map you have to move the cursor to the East).

You can increase the level of detail (zoom in) by clicking on the plus sign and you can decrease the level of detail (zoom out) by clicking on the minus sign.

Map Layers

There are two layers available:

  • The Base layer (always visible) detailing the geography features (Borders, cities, roads and their names).
  • The Bathymetry layer (hidden by default) detailing the depth of seas and oceans based on NOAA data.

You can select the visibility of the Bathymetry layer by clicking on the Layer control and then selecting the Bathymetry option from the dropdown menu:

Layer control dropdown menu.

Drawing Editing and Saving

To start drawing click on the pentagon symbol and you will see the following:

Drawing mode.

The mouse pointer shape will change to a cross and you will be able to add vertices by clicking on the map. You can change the zoom while you are drawing and you can also move the map by clicking and holding the left mouse button (see Exploring the map above).

After you have drawn at least three vertices you can decide to close your polygon or adding more vertices. To close your polygon simply click on the first vertex as hintend by the tooltip:

Drawing mode - drawing a polygon.

After closing the shape the Poly selector tool will revert to the default mode and you will see the polygon superimposed to the map. You will also see the coordinates (latitude and longitude) of the vertices on the bottom left and the sample code to import the polygon inside Bit.Sea on the bottom right.

The interface right after a polygon has been drawn.

Editing

To edit your polygon click on the pencil symbol:

Edit control.

You will be presented with the following interface:

Edit interface.

Clicking the left mouse on a square placed on a vertex, holding the button down and moving the cursor will move the corresponding vertex. The mouse cursor shape will change from an open hand to a pointing hand (see the figure above) to signal that you can drag the vertex. Clicking and dragging the middle point of a segment will create a new vertex that can be moved just as the other vertices.

When you are satisfied with your edit you can click on the "Save" button next to the edit control to save your changes and update the points. Clicking on cancel will undo all your changes.

The thrash can symbol allow you to delete a polygon by clicking on it. Just like the the edit control you have to press "Save" to confirm the deletion.

Saving the polygons

Clicking the "Select Text" button will highlight the Python code allowing you to copy that code into the clipboard.

Clicking the "Save All" button will save a file (named "polygon.py") containing the code. Please note that Poly selector displays only the points and the code of the latest drawn polygon but allows the drawing of multiple polygons. By clicking on the "Save All" button you will get all the drawn polygons saved inside the "polygon.py" file in the same order as they have been drawn.

Please note that Safari is known to be incompatible with the "Save All" function. Both Mozilla Firefox 38 or later and Google Chrome 48 or later are known to be compatible.
